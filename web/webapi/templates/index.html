      Tags

      All objects in BAS have 1 or more associated tags. You can reference tagged objects by prepending a string with a period, and then listing underscore-separated tags. For example, an exhaust air damper would be ".EXH_AIR_DMP" and a low light relay would be ".LO_REL". Specific objects can be referred to by prepending the name with '$', e.g. '$VAV 2'.

      • DIS : Discharge
      • AIR : Air
      • TMP : Temperature
      • SEN : Sensor
      • SPD : Speed
      • CMD : Point of Actuation
      • RET : Return
      • MIX : Mixed
      • ZON : Zone
      • SPT : Setpoint
      • OUT : Outside
      • DMP : Damper
      • HUM : Humidity
      • PRS : Pressure
      • FLW : Flow
      • POW : Power
      • CO2 : Carbon Dioxide
      • EXH : Exhaust
      • FAN : Fan
      • COO : Cooling
      • VLV : Valve
      • AHU : Air Handler
      • CCV : Cooling Coil
      • CWL : Cold Water Loop
      • HWL : Hot Water Loop
      • REL : Relay
      • HI : High
      • LO : Low
      • LIG : Light
      Geospatial

      You can additionally reference objects by their geospatial properties. Geospatial areas are referenced by prepending the name of an area with an exclamation point. For example, to see all areas on the fourth floor of Sutardja Dai Hall: "!Floor 4 < !Sutardja Dai"

      • Floors: !Floor 4
      • Buildings: !Sutardja Dai
      Sample Queries
      • Lights on 4th floor: .LIG < !Floor 4
      • All areas served by VAV 2: ! < $VAV 2
      Tips and Tricks

      Use the query strings '.' and '!' to get a list of all devices and geospatial areas, respectively. If a query doesn't work the first time, try typing an extra space after your full query. This happens because I am bad at javascript.
